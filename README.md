MD_openmm contains topology, parameter, and input files generated using the CHARMM-GUI web server for a membrane protein molecular dynamics system. The CHARMM-GUI default equilibration protocol is followed up to step 6.6, which represents the final restrained equilibration stage. The repository then provides an OpenMM production script for step 6.7, where a 20 ns unrestrained equilibration, equivalent to a production-level MD run, is performed. After this step, independent production-level molecular dynamics simulations can be carried out.

The OpenMM script is designed to be flexible for production MD workflows. It can continue simulations from different starting file formats, including AMBER-style restart files, OpenMM XML state files, OpenMM checkpoint files, and other supported coordinate formats. This makes the workflow useful for converting CHARMM-GUI-prepared systems into a more general OpenMM-based simulation pipeline.
